注釈インタフェースOption.Required

含まれているクラス:
Option
@Target(METHOD) @Inherited @Retention(CLASS) public static @interface Option.Required
オプションを必須オプションとしてマークします。

プリミティブ・タイプの必須オプションは、ビルダーを使用して構成する必要があります。 プリミティブ型のデフォルト値は無視されます。 このオプションは、デフォルト値の注釈と組み合せた場合には適用されません。たとえば、型がOptionalでないかぎり、フィールドは常に値を持ちます
