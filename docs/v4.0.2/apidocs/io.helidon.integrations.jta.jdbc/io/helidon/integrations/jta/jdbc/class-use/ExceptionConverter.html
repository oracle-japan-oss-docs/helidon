  • io.helidon.integrations.jta.jdbcでのExceptionConverterの使用

    io.helidon.integrations.jta.jdbcのコンストラクタとタイプExceptionConverterのパラメータ
    修飾子
    コンストラクタ
    説明
     
    JtaAdaptingDataSource(TransactionSupplier ts, TransactionSynchronizationRegistry tsr, boolean interposedSynchronizations, ExceptionConverter ec, DataSource ds, boolean immediateEnlistment)
    指定されたDataSourceをラップし、その「接続」がXAトランザクションに参加するのに役立つ新しいJtaAdaptingDataSourceを作成します。
     
    JtaAdaptingDataSource(TransactionSupplier ts, TransactionSynchronizationRegistry tsr, boolean interposedSynchronizations, ExceptionConverter ec, XADataSource xads, boolean immediateEnlistment, boolean closeXac)
    非推奨。
    このコンストラクタは、"借りる" XAConnectionへのエンド・ユーザー・コール元を許可し、close()メソッドを使用してエンド・ユーザー・コール元を"戻る"できるようにする特定のXA対応接続プールを処理するためにのみ存在します。これは、PooledConnection (XAConnectionの継承元)のドキュメントでは推奨されません。
