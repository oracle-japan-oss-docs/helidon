モジュール io.helidon.common
パッケージ io.helidon.common

反射された注釈インタフェース

@Retention(RUNTIME) @Target({TYPE,METHOD,FIELD,CONSTRUCTOR}) public @interface Reflected
この注釈が付いた型は、すべてのメソッドおよびフィールド(privateを含む)のリフレクション・サポートとともにネイティブ・イメージに追加されます。 このメソッドで注釈が付けられたメソッドまたはフィールドがリフレクション用に追加されます。

これは、Helidonに固有のGraalVMネイティブ・イメージreflect-config.jsonファイルにかわるものです。 この注釈の処理では、ネイティブ・イメージの構築時にクラスパスにio.helidon.integrations.graal:helidon-graal-native-image-extensionが必要です。

この注釈が付いたコンストラクタは、フィールドまたはメソッドのいずれかに(これはネイティブ・イメージのAPIの現在の制限です)の注釈が付いている場合にのみリフレクション用に追加されます。
