モジュール io.helidon.common.media.type

パッケージio.helidon.common.media.type.spi

パッケージio.helidon.common.media.type.spi
SPI:メディア・タイプへのリソースのカスタム・マッピングを提供します。

Javaサービス・ローダー・インタフェースMediaTypeDetectorに加えて、META-INF/media-types.propertiesでローカライズされたファイルをクラスパスに指定して、メディア・タイプ・マッピングをオーバーライドまたは追加することもできます。

プロパティ・ファイルでは、ファイル拡張子とメディア・タイプのマッピングのみがサポートされています。

例: 

 ooo=application/ooo
