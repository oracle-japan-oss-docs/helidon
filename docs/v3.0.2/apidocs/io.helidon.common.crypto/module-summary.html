モジュールio.helidon.common.crypto

モジュールio.helidon.common.crypto
Helidon共通暗号化ライブラリ。
このモジュールは、ダイジェスト(ハッシュ、hmac、シグネチャなど)を作成し、暗号化操作を実行してAES、ChaCha20またはRSAの使用でデータを暗号化および復号化するために使用されます。
デフォルトでは、サポートされているすべてのアルゴリズムが、対応するクラスで定数として使用可能です。 デフォルトで使用可能なアルゴリズムのみを使用する必要はありません。 すべてのアルゴリズムとそのプロバイダは、各クラスのビルダーで構成できます。
ダイジェスト: 
 HashDigest - class which creates hash digest of the message HmacDigest - class which creates message authentication code with addition of secret key Signature - class which creates RSA/EC signature
Encryption/Decryption: 
 SymmetricCipher - class which encrypts and decrypts provided message by symmetric cipher (AES, ChaCha20 etc.) AsymmetricCipher - class which encrypts and decrypts provided message by asymmetric cipher (RSA)
