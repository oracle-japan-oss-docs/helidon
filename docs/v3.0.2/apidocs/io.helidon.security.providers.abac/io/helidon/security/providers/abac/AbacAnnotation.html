モジュール io.helidon.security.providers.abac
パッケージ io.helidon.security.providers.abac

注釈インタフェースAbacAnnotation

@Retention(RUNTIME) @Target(ANNOTATION_TYPE) @Documented @Inherited public @interface AbacAnnotation
ABAC属性注釈のメタ注釈。 この注釈が付いた注釈は、ABACセキュリティ・プロバイダによって処理される必要があります(存在する場合)。存在しない場合は、認可を拒否する必要があります。 RolesAllowedは、デフォルトでメタ注釈付きとみなされます。
