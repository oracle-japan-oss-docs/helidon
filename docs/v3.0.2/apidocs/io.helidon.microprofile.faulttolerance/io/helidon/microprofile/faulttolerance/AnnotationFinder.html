クラスAnnotationFinder

java.lang.Object
io.helidon.microprofile.faulttolerance.AnnotationFinder
public class AnnotationFinder extends Object
指定されたJavaパッケージ内のインターセプタ・バインディングに関連付けられた推移的注釈を検索します。 これらの操作にはコストがかかる場合があるため、結果をキャッシュする必要があります。 たとえば、新しい注釈@TimedRetry自体は@Timeoutおよび@Retryで注釈が付けられ、メソッドはかわりに@TimedRetryを使用する場合は、両方を見つける必要があります。
