モジュール io.helidon.openapi

パッケージio.helidon.openapi

パッケージio.helidon.openapi
Helidon SE OpenAPIのサポート。

OpenAPISupportとそのBuilderを使用して、アプリケーションにOpenAPIのサポートを含めます。

Helidon SEではエンドポイントの識別に注釈処理を使用しないため、アプリケーションのOpenAPI情報を自分で指定する必要があります。 静的OpenAPI文書を提供することも、OpenAPI文書の作成に必要なデータを提供する独自のモデル処理クラスを実装および指定することもできます。
