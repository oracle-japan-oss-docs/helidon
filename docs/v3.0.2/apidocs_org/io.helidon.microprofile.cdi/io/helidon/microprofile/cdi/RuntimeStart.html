注釈インタフェースRuntimeStart

@Retention(RUNTIME) @Documented @Qualifier @Target({TYPE,METHOD,PARAMETER,FIELD}) public @interface RuntimeStart
アプリケーション(ビルド時間とは対照的)の実行時間。

これは、GraalVMネイティブ・イメージを使用する場合など、事前コンパイルの観点から見たものです。

アプリケーション・ライフサイクルには、次の2つのフェーズがあります:

  • BuildTimeStart - アプリケーションはどこにも接続しないでください。これは、native-imageが生成される前に環境を初期化するためです。 使用可能な構成はビルド時固有であり、このアプリケーションの実行時操作には使用できません
  • ランタイム(この注釈) - アプリケーションを起動しており、すべてのリソースを設定する必要があります。 実行時に使用できる構成は、このアプリケーションの実行時を対象としています

CDI Extensionの使用例: 

 void initRuntime(@Observes @RuntimeStart io.helidon.config.Config config) {}
ランタイムの終了はApplicationScopedの終了と同等であるため、追加の注釈を作成する必要はありません。
