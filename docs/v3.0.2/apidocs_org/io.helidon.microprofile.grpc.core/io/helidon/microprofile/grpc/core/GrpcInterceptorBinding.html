モジュール io.helidon.microprofile.grpc.core
パッケージ io.helidon.microprofile.grpc.core

注釈インタフェースGrpcInterceptorBinding

@Target(ANNOTATION_TYPE) @Retention(RUNTIME) @Documented public @interface GrpcInterceptorBinding
注釈型がgRPCインターセプタ・バインディング・タイプであることを指定します。 gRPCインターセプタ・バインディングは、gRPCクライアントまたはサーバー・インターセプタのターゲットgRPCサービスおよびメソッドへのバインディングを指定するために使用されます。

バインディングとしてマークされた注釈タイプは、サーバーgRPCインターセプタ実装クラス(jakarta.interceptor.Interceptor @Interceptor注釈でマークされたもの)のクライアントに適用して、その注釈をインターセプタに関連付ける必要があります。 その後、クラスまたはメソッドにアタッチするインターセプタを指定するために、jakarta.interceptor.Interceptors @Interceptors注釈のかわりに、またはそれに加えて注釈を適用できます。

関連付けられた注釈タイプは、TYPEまたはMETHOD(あるいはその両方)にのみ関連付ける必要があります。
