モジュール io.helidon.integrations.cdi.referencecountedcontext
パッケージ io.helidon.integrations.cdi.referencecountedcontext

注釈インタフェースReferenceCounted

@Documented @NormalScope(passivating=false) @Retention(RUNTIME) @Target({TYPE,METHOD,FIELD}) public @interface ReferenceCounted
注釈が付けられたBeanがReferenceCountedContextに属することを示す注釈。

ユース・ケース

次のすべてに該当する場合は、@ReferenceCountedでクラスに注釈を付けます:

  • クラスのインスタンスを1つのスレッドに関連付ける必要があります。
  • クラスの単一のスレッド固有のインスタンスを即時に破棄する必要がありますが、あまり再帰的ではありません。 具体的には、破棄されていない他のオブジェクトによって使用されなくなったインスタンスを破棄する必要があります。
  • クラスのインスタンスが破棄され、他のオブジェクトがクラスのインスタンスを要求する場合は、新しいインスタンスを指定する必要があります。

@ReferenceCounted-注釈付きクラスを考えるもう1つの方法は、わずかに短い「アプリケーション・スコープ」のメンバーであるかのように振舞うことです。

@ReferenceCounted-注釈付きクラスを考えたもう1つの方法は、「リクエスト・スコープ」のメンバーであるかのように動作しますが、最初のインスタンス化時に開始する名目リクエストがあることです。
